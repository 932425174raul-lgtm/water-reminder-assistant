喝水小助手

Windows 使用说明

从今天的一杯开始

少一点记录负担，
让喝水自然发生。

助手把重点放在适时提醒和一键记录。目标、杯量和提醒节奏都可以按医生建议与自己的生活安排调整。

01

在 Windows 安装

运行安装包

双击“喝水小助手 Setup 1.0.0.exe”，按照安装向导完成安装。安装后可从开始菜单或桌面快捷方式打开。

SmartScreen 提示

如果 Windows 显示“Windows 已保护你的电脑”，点击“更多信息”，确认文件来源后选择“仍要运行”。

设定今天

填写每日目标、常用杯容量、提醒间隔和清醒时段。清醒时段之外，助手不会打扰你。

02

记录一杯水

主界面

喝完常用的一杯水，点击“我喝了一杯”；也可以选择“先喝半杯”或用补记填写大概毫升数。

桌面胶囊

完整胶囊的“+1”会记录一杯。也可以在任意应用中按 Ctrl + Shift + W 快速记录。

不用精确

不必记录每一口。把常用杯容量设好，按杯记录就够了，目的不是做饮水账本。

03

使用桌面胶囊

精灵模式

水灵外圈显示当天进度，底部显示已喝毫升与目标。点击右上角展开图标，立即切换为完整胶囊。

完整显示

显示今日毫升、提醒文案、进度条和“+1”。胶囊任意位置都可以拖到你习惯的屏幕位置。

隐藏光条

右键选择“隐藏喝水胶囊”后，屏幕右侧会留下浅蓝光条。点击光条即可重新显示水灵。

右键胶囊、光条或系统托盘，可以切换三种显示方式、记录一杯水、打开助手或退出应用。

04

提醒与长期记录

柔性提醒

提醒出现后可以选择“我喝了”“5 分钟后”或“跳过本次”。跳过不会关闭下一次提醒。

饮品备注

无糖茶、碳酸、果汁、奶茶/果茶可作为备注记录，但不替代白水，也不计入喝水目标。

总结与导出

在“喝水总结”查看本周、本月、本季度和本年的达成情况。点击“导出 Excel”可得到每日与周期汇总。

有医生给出的饮水量或限水建议时，请始终以医生建议为准。
